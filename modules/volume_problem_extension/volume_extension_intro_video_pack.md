# Volume Problem Extension Intro Video Pack

Module: Volume Problem Extension

Source: `Y6-M-L10-L18-Study Book.pdf`, Lesson 16, PDF pages 109-125.

## Learning Outcome

After the intro, the learner should be able to turn a 3-D cube model into countable evidence: layers, removed cells, adjacent cubes, tunnels, height maps, front/left views, and displacement volume.

## Named Skills Shown Before Practice

- Layer Count
- Layer Map Sum
- Adjacent Cubes
- Tunnel Overlap
- Side Slice Tunnel
- Height Map
- Stair Stack
- Front & Left Views
- View Rebuild
- Reverse Missing
- Cube Displacement

## Scene Storyboard

| Scene | Title | Purpose | Child-facing voiceover |
|---|---|---|---|
| 1 | Slice The Solid | Show layers before counting. | "A hard 3-D shape becomes easier when we slice it into floors. Count each floor, then add." |
| 2 | Mark Missing Cubes | Make removed blocks visible. | "If a square is missing, mark it first. A 3 by 3 layer with one missing square has 8 cubes left." |
| 3 | Tunnels Can Overlap | Prevent double-subtraction. | "When two tunnels cross, the middle cube belongs to both tunnels. Subtract the tunnels, then fix the overlap." |
| 4 | Adjacent Cubes Still Count | Stop visible-face counting. | "When cube towers touch, some faces disappear, but the cubes are still there. Add every tower." |
| 5 | Read Height Maps | Teach plan numbers as towers. | "A number in a top-view square tells how many cubes are stacked there. Add the numbers." |
| 6 | Stairs Are Layers | Count stepped solids cleanly. | "A staircase is 1, then 2, then 3 cubes in a slice. Use the triangular total, then stretch it." |
| 7 | Views Are Clues | Combine front, left, and plan views. | "One view is not enough. Front gives column heights, left gives row heights, and plan shows where cubes can sit." |
| 8 | View Direction Matters | Practise front and left views directly. | "For the front view, take columns. For the left view, take rows. The direction changes the answer." |
| 9 | Reverse The Count | Work backwards from volume. | "If the total is given, divide by layers first, then compare with the full layer." |
| 10 | Cubes Move Water | Link cube count to displacement. | "If each cube is 1 cubic centimetre, the cube count is the volume that pushes water up." |
| 11 | Ready To Rebuild | Map intro to practice. | "Now train with fresh adjacent cubes, layer plans, tunnels, views, stairs, and water extensions." |

## Warmup Gate

1. Count a 3 by 3 layer with one missing square.
2. Add three layer counts.
3. Find the overlap correction for two crossing tunnels.
4. Add a small adjacent-cube plan.
5. Read one front view or left view from a tower plan.

## Misconception Risks

- Counting visible cubes only and forgetting hidden cubes.
- Subtracting crossing tunnels twice at their shared cube.
- Losing cubes because adjacent faces are hidden.
- Reading plan-view height numbers as labels instead of tower heights.
- Mixing up front-view columns with left-view rows.
- Trusting one view without checking the plan.
- Using total tank height instead of displaced volume divided by base area.

## NotebookLM Steering Prompt

Create a short explainer for an 11-year-old learning Volume Problem Extension. Use the supplied Lesson 16 pages. Start with the idea that a difficult cube solid can be sliced into layers. Teach how to mark missing cells, count each layer, add layer plans, count adjacent cube towers even when faces are hidden, and handle tunnels that overlap. Then introduce plan-view height maps, staircase solids, direct front-view and left-view reading, and front-left-plan view reconstruction. End by connecting counted unit cubes to cubic-centimetre volume for water displacement. Keep every example visual, concrete, and based on fresh numbers rather than memorised page diagrams.
