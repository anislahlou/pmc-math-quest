# Triangles With Equal Height - Intro Video Pack

## Learning Outcome

By the end of the intro, the learner should be able to say:

> If triangles have the same perpendicular height, their areas change in the same ratio as their bases. If the base is also the same, the areas are equal.

The intro deliberately shows the height before the formula. The area rule appears only after the child has seen the perpendicular drop and the half-rectangle model.

## Storyboard

1. **Same Height Spotter**
   - Visual: one triangle with sloping side and perpendicular height.
   - Narration: height is the straight drop to the base.
   - Skill: Height Detective.

2. **Half Rectangle**
   - Visual: triangle inside a matching rectangle.
   - Narration: a triangle is half of a matching rectangle.
   - Skill: Half Rectangle.

3. **Base Ratio Mirror**
   - Visual: two triangles between the same parallel lines.
   - Narration: the shared 1/2 x height part cancels in comparison.
   - Skill: Base Ratio Mirror.

4. **Sliding Apex Twin**
   - Visual: two different-looking triangles with same base and height.
   - Narration: shape changes, area does not.
   - Skill: Sliding Apex Twin.

5. **Double Then Divide**
   - Visual: area and one dimension given, missing dimension marked.
   - Narration: reverse the formula by doubling first.
   - Skill: Double Then Divide.

6. **Split Base Shares**
   - Visual: common apex over a split base line.
   - Narration: one apex gives one shared height.
   - Skill: Split Base Shares.

7. **Area Order Line**
   - Visual: three triangles sharing one height.
   - Narration: area order follows base order.
   - Skill: Area Order Line.

8. **Shaded Difference**
   - Visual: a shaded gap between same-height triangles.
   - Narration: subtract bases first, then use the shared height.
   - Skill: Shaded Difference.

9. **Expression Area Chase**
   - Visual: base labels with x expressions.
   - Narration: equal-height area facts become equations.
   - Skill: Expression Area Chase.

10. **Ready To Train**
    - Visual: quick sweep of all problem types.
    - Narration: practice will test area, ratios, reverse questions, split bases, diagrams, shading, and algebra.
    - Skill: full module recap.

## Audio Requirement

The live module uses bundled `.wav` narration files for all ten scenes. Browser speech synthesis is kept as a fallback only. Release QA blocks if any scene is missing an audio file or if the file is too small to be a real narration asset.

## Warmup Gate

Before practice unlocks, the child answers four warmup checks:

- identify the real height;
- map base ratio to area ratio;
- reverse the area formula in the right order;
- recognise equal area from same base and parallel height.

This gate is part of the learning design, not a quiz for points. It prevents the exercises from starting before the diagrams are meaningful.
