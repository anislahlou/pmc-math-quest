# PMC Math Quest - GitHub Pages Ready

This folder is ready to publish as a static GitHub Pages site.

## Launch

The root `index.html` redirects to:

`run/pmc_simulation_app.html`

## Recommended GitHub Pages Setup

1. Create a public repository on GitHub, for example `pmc-math-quest`.
2. Upload every file and folder in this directory to the repository root.
3. In GitHub, open `Settings > Pages`.
4. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save. GitHub will provide the live web address after the first Pages build.
