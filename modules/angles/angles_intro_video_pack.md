# Angles Intro Video Pack

Date: 2026-04-29

Audience: 11-12 year olds preparing for PMC/JMC-style angle chasing.

Length target: 120-150 seconds.

Primary source anchor: Lesson 3, "Angles", PDF pages 40-53 / book pages 30-43.

## Core Learning Outcome

The learner understands that angle problems are solved by naming the correct move, then chaining moves carefully. No guessing, no measuring; every step is justified by one of the named angle facts on the toolkit card.

Core moves:

```text
straight line -> 180°
triangle -> 180°
isosceles -> equal base angles
parallel Z/F -> equal
parallel C -> adds to 180°
parallel zigzag -> preserve a + b targets
equilateral chase -> write in 60° first
polygon sum -> (n - 2) x 180°
regular polygon -> divide the interior sum by n
```

## Named Skills

The learner sees this skill card before any scene starts. Each skill maps to a registry token used by the practice bank:

- Triangle total - missing angles in a triangle add to 180 degrees.
- Z/F/C angles - the three parallel-line moves: Z-angles are equal, F-angles are equal, C-angles add to 180 degrees.
- Zigzag totals - parallel-line zigzag diagrams where the target is `a + b`, not a single angle.
- Angle chase - chaining two or three of the named moves through a competition diagram.
- Polygon formula - interior-angle sum is `(n - 2) x 180`, and one corner of a regular polygon divides that total by `n`.

## What The Child Should Be Able To Do After The Intro

The learner should be able to:

1. Find a missing angle in a triangle using the triangle total.
2. Use isosceles equal-side marks to make equal base angles.
3. Convert exterior or straight-line angles into interior angles.
4. Recognise alternate Z-angles and corresponding F-angles in parallel-line diagrams.
5. Use co-interior C-angles as a `180°` pair within the Z/F/C angles family.
6. Solve parallel-zigzag questions where the target is `a + b`, using zigzag totals.
7. Chain multiple angle facts in a competition-style angle chase diagram.
8. Use hidden `60°` angles in equilateral-triangle chases.
9. Find a polygon interior-angle sum using the polygon formula `(n - 2) x 180°`.
10. Find one regular polygon interior angle by dividing the total by `n`.

## Warmup Gate

Before practice unlocks, the child answers four warmup checks:

1. Find the missing angle in a triangle whose other two angles are 48 degrees and 67 degrees.
2. Two parallel lines are cut by a transversal. One Z-angle is 64 degrees. What is the matching Z-angle?
3. A zigzag between parallel lines has corners `a = 40` and `b = 55` inside the bends. What is `a + b`?
4. A regular hexagon has six equal interior angles. What is each angle?

These four questions force the learner to name the move (triangle total, Z/F/C angles, zigzag totals, polygon formula) before calculating.

## Misconception Risks

- Treating C-angles as if they were equal, the way Z and F angles are.
- Adding only a single angle to find `a + b` in a zigzag totals question.
- Forgetting that exterior angles on a straight line must be converted before using the triangle total.
- Using `(n - 2) x 180` and then forgetting to divide by `n` for a regular polygon corner.
- Placing an angle label inside the wrong sector of an angle, especially with supplementary pairs.

## Full Voiceover Draft

Angles can look messy, but the trick is calm: do not guess. Name the move you are about to use, then calculate the angle, then pass that angle on to the next step. This intro walks through every named move on the toolkit card, so practice problems become a series of named moves rather than a guessing game.

There are simple moves in this chapter, and then challenge moves built from them. First, a straight line totals 180 degrees. Second, the three angles inside a triangle total 180 degrees. Third, in an isosceles triangle, matching sides make matching base angles, so the two base angles share whatever is left after the top angle.

Parallel-line questions use the Z/F/C angles family. Z-angles are equal, F-angles are equal, and C-angles add to one hundred eighty degrees. Practice will mix all three, so name which one you are using before you write a number.

In harder parallel-line questions, the answer might be a plus b, not just one angle. These are zigzag totals questions. Keep both target angles in view, then collect the turns that feed them. The zigzag answer is always the sum of the alternate angles you can identify on each parallel line.

Now watch how all this becomes an angle chase. If an outside angle is shown, first turn it into the inside angle using a straight line. Then use a triangle total, or pass an equal angle through parallel lines. Competition questions often hide the answer two or three moves away.

If you see an equilateral triangle, write in sixty degrees at all three corners first. Those hidden sixty-degree corners often unlock the chase by telling you exactly how the surrounding angles have turned.

For polygons, use the polygon formula. A polygon with n sides can be split into n minus two triangles. So the total interior angle sum is `(n - 2) x 180`. If the polygon is regular, all the angles are equal, so divide that total by n for one corner.

In practice, say the name of the move before calculating. That keeps the diagram under control and stops you from chasing the wrong angle.

## Acceptance Gate

The intro is blocked if:

- it starts with formulas before naming visual angle moves
- it mixes up equal parallel angles with co-interior supplementary angles
- it does not show straight-line conversion before triangle solving
- it does not include at least one competition-style angle chase
- it misses expression targets such as `a + b` when they appear in the source
- it does not include polygon interior sum `(n - 2) x 180°`
- any angle label floats away from the exact angle it describes
