# Triangles With Equal Height - Intro Video Pack

Mission: same-height triangles, area ratios, reverse area, and algebra targets.

Audience: 11-12 year olds preparing for PMC-style equal-height problems.

Length target: 120-150 seconds.

## Learning Outcome

By the end of the intro, the learner should be able to say:

> If triangles have the same perpendicular height, their areas change in the same ratio as their bases. If the base is also the same, the areas are equal.

The intro deliberately shows the height before the formula. The area rule appears only after the child has seen the perpendicular drop and the half-rectangle model. Reverse questions and algebra targets are introduced after the visual rule is settled, never before.

## Named Skills

The learner sees this skill card before any scene starts. Each skill maps to a registry token used by the practice bank:

- Height detective - choose the perpendicular drop, not the sloping side or a decorative label.
- Area ratios - same height means area ratio mirrors the base ratio.
- Split bases - one apex above a split base line shares one height, so the base pieces split the area.
- Reverse formula - to find a missing base or height from a known area, double first, then divide.
- Algebra targets - turn equal-height area facts into equations in `x`, `a`, `b`, or `a + b`, then answer the exact target the question asks for.

## Storyboard

1. **Same Height Spotter**
   - Visual: one triangle with a sloping side and a perpendicular drop, the right-angle mark highlighted at the foot of the height.
   - Narration: height is the straight drop to the base, never a slanted side.
   - Skill: Height detective.

2. **Half Rectangle**
   - Visual: a triangle nested inside a matching rectangle of the same base and height.
   - Narration: a triangle takes half of the matching rectangle, so area is base times height divided by two.
   - Skill: Half Rectangle area rule.

3. **Base Ratio Mirror**
   - Visual: two triangles between the same parallel lines, with bases highlighted as `2` and `5`.
   - Narration: the shared half-times-height part cancels in the comparison, so the area ratio mirrors the base ratio.
   - Skill: Area ratios.

4. **Sliding Apex Twin**
   - Visual: two different-looking triangles sharing the same base and the same parallel line above.
   - Narration: the shape changes, but the perpendicular height does not, so the area does not change either.
   - Skill: Area ratios with equal areas.

5. **Double Then Divide**
   - Visual: an area card `36 cm^2` and a height `9 cm`. The area doubles to `72`, then divides by `9` to reveal base `8`.
   - Narration: reverse formula in the right order. Because the area rule divides by two, undo that first by doubling, then divide by the known length.
   - Skill: Reverse formula.

6. **Split Base Shares**
   - Visual: one apex above a base line split into two segments `3` and `5`. The two smaller triangles share the same dropped height.
   - Narration: one apex above one straight base line gives one shared height, so the base pieces split the area in the same ratio.
   - Skill: Split bases.

7. **Area Order Line**
   - Visual: three triangles standing between the same two parallel lines, bases of length `4`, `6`, `9`.
   - Narration: with equal height, the area order follows the base order. Long base, large area; short base, small area.
   - Skill: Area ratios in ordered form.

8. **Shaded Difference**
   - Visual: a shaded gap between two same-height triangles, the smaller base subtracted from the larger.
   - Narration: subtract bases first, then multiply by the shared height, then halve. Do not compute each triangle separately.
   - Skill: Split bases by subtraction.

9. **Expression Area Chase**
   - Visual: two triangles with bases labelled `x + 3` and `2x - 5`, sharing the same height. An equals sign sets the matching base expressions equal.
   - Narration: turn the equal-height area fact into an equation, then read the question carefully to see whether it wants `x`, a base, or `a + b`.
   - Skill: Algebra targets.

10. **Ready To Train**
    - Visual: a quick sweep of all problem types - area, ratios, reverse questions, split bases, diagrams, shading, algebra.
    - Narration: every practice problem fits one of these named moves. Name the move before you write a number.
    - Skill: full module recap, including reverse formula and algebra targets.

Each storyboard scene maps to one named skill in the practice bank through its `classicId`. The module test blocks release if a registry skill is tested but absent from the intro.

## Warmup Gate

Before practice unlocks, the child answers four warmup checks:

- identify the real height in a triangle picture (perpendicular drop, not the slanted side);
- map a base ratio of `2:5` to the area ratio of two same-height triangles;
- reverse the area formula in the right order (double the area first, then divide by the known length);
- recognise equal area from same base and same parallel height.

This gate is part of the learning design, not a quiz for points. It prevents the exercises from starting before the diagrams are meaningful.

## Misconception Risks

- Labelling a sloping side as the height because it looks longer in the picture.
- Squaring or doubling the base ratio when the height is already shared.
- Reversing the area formula by dividing first, which leaves a fraction the child mishandles.
- Solving for `x` and forgetting that the question asks for `a + b` or for one of the bases instead.
- Treating a shaded gap as two separate area calculations rather than one base-difference shortcut.

## Voiceover Draft

The first job of this intro is to train the eye. A triangle height is not the sloping side. It is the straight perpendicular drop to the base, marked with a small square corner. Once you can spot that height, the rest of the lesson stays calm.

Next, place a triangle inside a matching rectangle. Same base, same height. The triangle fills exactly half of the rectangle, so the area rule is base times height divided by two. The visual reason comes before the formula.

Now the key move. If two triangles have the same height, both areas contain the same half-times-height factor. That shared factor cancels from any comparison, so the area ratio mirrors the base ratio. Bases in the ratio `2:5` give areas in the ratio `2:5`. No squaring, no doubling, no swapping.

If two triangles share the same base and sit between the same parallel lines, the perpendicular height does not change as the apex slides. Different shapes, same area. The sliding apex twin is the surprise that proves the rule.

Reverse formula questions give the area and ask for a missing base or height. Because the area rule divides by two, you undo that division first. Double the area, then divide by the known length. That order keeps the arithmetic clean.

Split bases use one apex above a straight base. Every smaller triangle in the picture shares one perpendicular height. So the base pieces split the area in the same ratio as the base segments themselves.

Area Order Line ranks several same-height triangles by base size. Shaded Difference subtracts a smaller same-height triangle from a larger one by subtracting bases first.

Algebra targets turn the equal-height area fact into an equation. The bases might be labelled `x + 3` and `2x - 5`. Set the matching pieces equal, solve for `x`, then re-read the question to check whether it wants `x`, a base, or `a + b`. The most common slip is solving for `x` and forgetting the actual target.

By the end of this intro, every practice problem will fit one of the named moves: height detective, area ratios, split bases, reverse formula, or algebra targets. Name the move before you write a number.
