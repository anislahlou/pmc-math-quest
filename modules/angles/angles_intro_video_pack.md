# Angles Intro Video Pack

Date: 2026-04-29

Audience: 11-12 year olds preparing for PMC/JMC-style angle chasing.

Length target: 90-120 seconds.

Primary source anchor: Lesson 3, "Angles", PDF pages 40-53 / book pages 30-43.

## Core Learning Outcome

The learner understands that angle problems are solved by naming the correct move, then chaining moves carefully.

Core moves:

```text
straight line -> 180°
triangle -> 180°
isosceles -> equal base angles
parallel Z/F -> equal
parallel C -> adds to 180°
parallel zigzag -> preserve a + b targets
equilateral chase -> write in 60° first
polygon sum -> (n - 2) x 180°
regular polygon -> divide the interior sum by n
```

## What The Child Should Be Able To Do After The Intro

The learner should be able to:

1. Find a missing angle in a triangle.
2. Use isosceles equal-side marks to make equal base angles.
3. Convert exterior/straight-line angles into interior angles.
4. Recognise alternate Z-angles and corresponding F-angles.
5. Use co-interior C-angles as a `180°` pair.
6. Solve parallel-zigzag questions where the target is `a + b`.
7. Chain multiple angle facts in a competition-style diagram.
8. Use hidden `60°` angles in equilateral-triangle chases.
9. Find a polygon interior-angle sum using `(n - 2) x 180°`.
10. Find one regular polygon interior angle by dividing the total by `n`.

## Full Voiceover Draft

Angles can look messy, but the trick is calm: do not guess.

Name the move.

There are simple moves in this chapter, and then challenge moves built from them.

First, a straight line is 180 degrees.

Second, the angles inside a triangle add to 180 degrees.

Third, in an isosceles triangle, matching sides make matching base angles.

Fourth, with parallel lines, Z-angles are equal.

Fifth, F-angles are also equal.

Sixth, C-angles are not equal. They add to 180 degrees.

In harder parallel-line questions, the answer might be a plus b, not just one angle.

Keep both target angles in view, then collect the turns that feed them.

Now watch how this becomes an angle chase.

If an outside angle is shown, first turn it into the inside angle using a straight line.

Then use a triangle total, or pass an equal angle through parallel lines.

Competition questions often hide the answer two or three moves away.

If you see an equilateral triangle, write in sixty degrees first.

Those hidden sixty-degree corners often unlock the chase.

For polygons, use the triangle-splitting trick.

A polygon with n sides can be split into n minus two triangles.

So the total interior angle sum is:

```text
(n - 2) x 180
```

If the polygon is regular, all the angles are equal, so divide that total by n.

In practice, say the name of the move before calculating.

That keeps the diagram under control.

## Acceptance Gate

The intro is blocked if:

- it starts with formulas before naming visual angle moves
- it mixes up equal parallel angles with co-interior supplementary angles
- it does not show straight-line conversion before triangle solving
- it does not include at least one competition-style angle chase
- it misses expression targets such as `a + b` when they appear in the source
- it does not include polygon interior sum `(n - 2) x 180°`
- any angle label floats away from the exact angle it describes
